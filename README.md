# Abdirahman Warsame - Portfolio Website

A modern, responsive portfolio website showcasing my full-stack development skills and projects.

## 🚀 Tech Stack

- **Frontend**: React 18, JavaScript (ES6+)
- **Styling**: Tailwind CSS
- **Routing**: React Router v6
- **Build Tool**: Vite
- **Email Service**: EmailJS

## 💼 Features

### Multi-Page Navigation
- **Home**: Scrollable landing page with hero, services, about, skills preview, and featured projects
- **About**: Personal story and professional background
- **Skills**: Comprehensive technology showcase with proficiency levels
- **Projects**: Portfolio of completed and in-development projects with GitHub links
- **Experience**: Education, certifications, and ongoing projects
- **Pricing**: Transparent pricing for web development services (Kenyan market rates)
- **Contact**: Working contact form with EmailJS integration + direct contact links

### Design Highlights
- Fully responsive design (mobile, tablet, desktop)
- Modern gradient backgrounds and color schemes
- Smooth hover effects and transitions
- Sticky navigation with mobile menu
- Professional card-based layouts

### Projects Showcase
- Art Gallery Website (Frontend)
- Restaurant Ordering System (Full-Stack: React + Flask)
- Beauty Shop Platform (Full-Stack: React + FastAPI)
- Hospital Management System (In Development)
- Dates & Spices E-commerce (In Development)

## 🛠️ Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📧 Contact Form Setup

The contact form uses EmailJS. To enable it:

1. Create account at https://www.emailjs.com/
2. Set up email service and template
3. Update credentials in `src/pages/Contact.jsx`

See `EMAILJS_SETUP.md` for detailed instructions.
